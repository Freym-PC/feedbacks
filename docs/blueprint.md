# **App Name**: WorkRecs

## Core Features:

- User Accounts: User authentication and profile creation to manage professional details and preferences.
- Recommendation Input: Form to allow users to input and categorize their service recommendations, with a simple text field and category selector.
- Recommendation Feed: Display service recommendations based on the user's selected professional sector.
- Feedback Summarizer: AI tool to summarize user feedback.

## Style Guidelines:

- Primary color: #000000 to inspire confidence and reliability.
- Background color: #FFFFFF for a clean, professional backdrop.
- Accent color: #CECECE for positive feedback and calls to action.
- Clean and modern fonts to ensure readability.
- Simple, intuitive icons representing different professional sectors.
- Clean and intuitive layout, focused on user profiles and personalized content.
